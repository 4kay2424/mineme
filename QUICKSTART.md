# Quick Start Guide - Multi-Coin Web Miner

## Installation (30 seconds)

1. **Extract the ZIP file** to any folder on your computer
2. **Open `index.html`** in your web browser (double-click or right-click → Open with Browser)
3. That's it! The miner is ready to use.

## First Mining Session (2 minutes)

### Step 1: Select a Coin
- Click the **"Select Coin"** dropdown
- Choose from:
  - **DuinoCoin (DUCO)** - Easiest to start
  - **CoinIMP** - Medium difficulty
  - **MintMe** - Pre-configured CryptoNight
  - **SHA-256** - Bitcoin-compatible
  - **CryptoNight** - Custom configuration
  - **WebMinePool** - Alternative pool

### Step 2: Enter Your Wallet
- Paste your wallet address or username in the **"Wallet / Username"** field
- Example: `your_wallet_address_here`

### Step 3: Adjust Settings (Optional)
- **CPU Throttle**: Drag slider to control CPU usage
  - Left (0%) = Maximum speed, maximum CPU usage
  - Right (100%) = Minimum speed, minimal CPU usage
- **For MintMe**: Pool URL is pre-filled; optionally change algorithm

### Step 4: Enable & Start
- ✅ Check the **"I understand and consent..."** checkbox
- Click **"▶️ Start Mining"** button
- Watch the statistics update in real-time!

### Step 5: Monitor
- **Hashrate**: Your mining speed (H/s, KH/s, MH/s)
- **Accepted Shares**: Valid work submitted
- **Mining Time**: How long you've been mining
- **Console Log**: Real-time activity messages

### Step 6: Stop
- Click **"⏹️ Stop Mining"** to stop anytime
- Your browser will return to idle state

## MintMe Mining (Recommended for Beginners)

MintMe is pre-configured and easiest to start:

1. Select **"MintMe (CryptoNight)"**
2. Enter your MintMe wallet address
3. Check consent checkbox
4. Click Start!

The pool, algorithm, and all settings are pre-configured. Just provide your wallet and go.

## Tips for Best Results

### Maximize Hashrate
- Set throttle to **0%** for maximum speed
- Close other applications
- Use a wired internet connection
- Ensure good CPU cooling

### Reduce System Impact
- Set throttle to **50-80%** for balanced use
- Keep browser tab visible (some browsers throttle background tabs)
- Monitor CPU temperature

### Troubleshooting

**Mining won't start?**
- ✓ Consent checkbox must be checked
- ✓ Wallet field must not be empty
- ✓ Try refreshing the page (F5)

**Low hashrate?**
- ✓ Reduce throttle slider (move left)
- ✓ Close other CPU-intensive applications
- ✓ Check if browser tab is active

**Browser freezing?**
- ✓ Increase throttle slider (move right)
- ✓ Reduce number of tabs open
- ✓ Try a different browser

## Customization

### Change Pool (CryptoNight/MintMe)
1. Select "CryptoNight" or "MintMe"
2. In "Pool URL" field, enter: `pool.example.com:port`
3. Example: `pool.mintme.com:3333`

### Change Algorithm (CryptoNight)
1. Select "CryptoNight"
2. Choose from dropdown:
   - **cn-fast**: Fastest
   - **cn-half**: Medium
   - **cn-turtle**: Slowest but most memory-intensive
   - **cn-upx2**: UltraPlus variant
   - **randomx**: RandomX algorithm
